					Команды терминала Linux



 pwd 		- 	Посмотреть где я
 

 mkdir 		- 	Создать папку

 
 Clear		-	Очистить консоль


 ls		-	список файлов в папке
 

 ls -la		-	список файлов, включая скрытые
 

 cd foldername/	-	Зайти в папку 
 

 mkdir namefolder1 namefolder2 namefolder3 		-	Создать 3 папки
 

 cd foldername	-	Зайти в любую папку
 

 touch file1.txt file2.txt file3.txt file1.json file2.json	-	Создать 5 файлов (3 txt, 2 json)
 

 vim filename.txt	-	 Открыть любой txt файл
 

 I (Insert)	-	написать туда что-нибудь, любой текст.
 

 :WQ		-	сохранить и выйти.
 

 cd ..		-	Выйти из папки на уровень выше

 mv filename1 filename2 foldername	-	переместить любые 2 файла, которые вы создали, в любую другую папку.
 

 cp filename1 filename2 foldername	-	скопировать любые 2 файла, которые вы создали, в любую другую папку.
 

 find . -name filename		-	Найти файл по имени
 

 cat filename.txt	-	просмотреть содержимое в реальном времени
 

 head filename.txt	-	вывести несколько первых строк из текстового файла
 

 tail /var/log/syslog filename	-	вывести несколько последних строк из текстового файла
 

 less filename.txt	-	просмотреть содержимое длинного файла
 

 date    time		-	вывести дату и время
